# OCFA

OCFA は、オリジナルキャラクターを描いて登録し、イベントやファンアートを通じて交流できるWebサイトです。

サイト内でイラストを描き、下書きとして保存し、気に入った絵をキャラクターとして登録できます。  
登録したキャラクターはキャラファイルとして公開でき、イベントに参加させたり、ほかの参加キャラへファンアートを描いたりできます。

## Concept

OCFA は「自分のオリキャラを置いておける場所」と「誰かのキャラを見て描ける場所」をひとつにしたサイトです。

SNSのように数字や拡散を中心にするのではなく、キャラクターを登録し、イベントを通してゆるく見たり描いたりできる場所を目指しています。

## Fan Art Flow

OCFAのファンアート機能では、イベントに参加しているキャラクターを見ながらファンアートを描けます。

基本的な流れは以下です。

1. ユーザーがイベントにキャラクターを参加させる
2. ほかのユーザーが参加キャラの「この子を描く」を選ぶ
3. 描く宣言が作成される
4. キャラクター画像やプロフィール、注意事項を見ながら描く
5. ファンアートを保存する
6. イベント詳細に投稿されたファンアートが表示される

描く宣言は描いた本人だけが確認でき、描かれる側には表示されません。  
ただし、描く宣言中または投稿済みのファンアートがある場合、そのキャラクターのイベント参加は取り消せないようになっています。

## Character Rules

イベントに参加できるキャラクターは、以下の条件を満たすものです。

- 公開中であること
- ファンアート歓迎であること
- 削除されていないこと
- ログイン中の本人が所有していること

## Design

OCFAは、紙・スケッチブック・アトリエのような雰囲気を意識したデザインです。  
やわらかい色、角丸のカード、落ち着いた背景を使い、創作キャラクターを置きやすい空気感を目指しています。
