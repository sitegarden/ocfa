OC Drawing Game

OC描き合いゲーム

みんなで自分のOCを描いて、シャッフルして、ファンアートを描き合うゲームです。

How to Play

遊び方

  1. オーナーが部屋を作ります。
  2. 参加者が集まったらゲーム開始。
  3. まずは時間内に自分のOCを描きます。
  4. その後、順番にほかの参加者のOCへファンアートを描きます。
  5. 全員分のターンが終わったら、作品をいっせいに公開します。

Point

3人なら9枚できる

3人で遊ぶ場合、それぞれのOCに対して3人分の絵が並びます。 自分のOCも、みんなのFAもまとめて見られます。

Create Room

部屋を作る

部屋作成はログイン中のユーザーのみできます。 作成した部屋URLを共有すると、ゲストも名前を入れて参加できます。
